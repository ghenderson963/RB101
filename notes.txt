Truthiness - we can use any expression in a conditional, or with logical operators, and as long as it 
  doesn't evaluate to false or nil, it is considered true 

variables initialed inside of an if statement can be used outside the "block"

pseudo code 
keyword	meaning
START -	start of the program
SET	- sets a variable we can use for later
GET	- retrieve input from user
PRINT	- displays output to user
READ -	retrieve value from variable
IF / ELSE IF / ELSE -	show conditional branches in logic
WHILE	- show looping logic
END -	end of the program
